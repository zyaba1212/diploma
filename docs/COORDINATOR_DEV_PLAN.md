## Чеклист перед тем как писать код в чате координатора

1. Задача уже разложена по ролям в **`docs/agents/*-prompts.md`**, **`docs/ARCHITECT_CHAT_PLAN.md`** или **`AGENTS.md`**? → **Не** реализовывать схему + API + UI целиком самому; оформить/обновить промпт и отдать **ролевому агенту**.
2. Это одна точечная правка / hotfix / приёмка после агента? → можно править код в рамках правил `.cursor/rules/coordinator-architect.mdc`.
3. Владелец явно сказал «делай сам координатору»? → можно.

Если п.1 **да**, а ты всё равно пишешь весь объём — **нарушение процесса**.
