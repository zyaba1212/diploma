# Design

## Визуальные правила (минимум)

- Кабели: линии, цвет зависит от типа (fiber/copper).
- Спутники: точки на орбите, по умолчанию нейтральный цвет.
- Базовые станции/оборудование: точки на поверхности, иконка/цвет по типу.

## UI

- Поверх сцены — компактная панель управления (режим 3D/2D, фильтры, авторизация).
- Чёткие состояния: loading/error/empty.
